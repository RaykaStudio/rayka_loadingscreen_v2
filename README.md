# RAYKA Loading Screen V2

RAYKA Loading Screen V2 comes with a completely redesigned interface and a smoother user experience.

## ✨ What's New?

* Fully redesigned modern UI.
* Added a **Music Manager** for easier music customization.
* Added an **Announcement System**.
* Support for both **Dark Mode** and **Light Mode** themes.
* Smooth and optimized animations.
* Ability to add your own social media links directly to the loading screen.
* Clean and easy-to-edit structure.

## 📂 Customization

### 🎵 Change or Add Music

To change the default music or add your own tracks, edit:

```js
script.js
```

### 🖼️ Change Texts and Images

To customize texts, images, and general content, edit:

```html
index.html
```
